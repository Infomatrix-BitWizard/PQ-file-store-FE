# Init

- create default next app

- install electron
```shell
npx i -g @electron-forge/cli
```

- create electron app
```shell
electron-forge init electron-app
```

---

# Install

```shell
npm i
cd electron-app
npm i
```

---

# Run

- run next app
```shell
npm run dev
```

- run electron app (in new shell)
```shell
cd electron-app
npm run start
```
